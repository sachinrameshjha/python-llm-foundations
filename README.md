# 🐍 Python Foundations — Study · Hide · Write

A self-paced Python learning track built around **active recall** — not passive reading.  
Every concept follows the same 3-block pattern: **Study → Hide → Write from memory.**

Built by [@sachinrameshjha](https://github.com/sachinrameshjha) as part of a Python → LLM Engineering learning sprint.

## 🗺️ What's Covered

### Session 1 — Variables, Types, f-strings
- Variable assignment, reassignment, multi-assignment
- `type()`, type casting (`int()`, `str()`, `float()`)
- f-string formatting including `{expr}` and `:.2f`

### Session 2 — Lists, Loops, List Comprehensions
- List creation, indexing, slicing, `.append()`, `.pop()`
- `for` loop, `range()`, `enumerate()`
- List comprehensions: transform, filter, combined

### Session 3 — Dictionaries, Nested Dicts, `.get()`
- Dict creation, access, add/update/delete keys
- `.get()`, `.keys()`, `.values()`, `.items()`
- Nested dicts, nested access, looping with calculations

### Session 4 — Functions, Return, Lambda, `max(key=)`
- Defining functions, `return` values, `None`-returning functions
- Default parameters, keyword arguments
- `lambda` functions, `max()` and `min()` with `key=lambda`

### Session 5 — try/except, Error Types, Defensive Coding
- `try/except` basics, `as e` for error inspection
- Catching `KeyError`, `IndexError`, `TypeError` specifically
- Guard clauses, `.get()` with defaults

### Session 6 — Classes, `__init__`, `self.`, Methods
- Defining a class, `__init__`, instance attributes
- `self.` — the difference between instance and local variables
- Writing methods, calling methods from within other methods

## 🧠 The Study → Hide → Write Method

Each concept has three cells:

1. **📖 STUDY** — Read a clean, minimal example with inline comments
2. **🙈 HIDE** — Plain English description of what you just read (no code)
3. **✍️ WRITE** — Recreate the concept from memory in a blank cell

The goal is to be able to write every pattern from scratch — not to recognise it.
